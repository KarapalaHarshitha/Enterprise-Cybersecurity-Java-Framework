# Enterprise Multi-Threaded CyberSecurity Framework (Java Core)
An asynchronous, multi-threaded connection monitoring and data-integrity policing system developed natively within the BlueJ IDE environment.

## ⚙️ Architectural Subsystems
1. **Asynchronous Network Gateway (`NetworkInspector`)**: Implements the `Runnable` interface to intercept inbound packet structures on isolated parallel CPU tracks, ensuring thread-safe execution loops.
2. **Defensive Connection Filter (`Firewall Check`)**: Utilizes array-lookup algorithms to cross-reference runtime IP packets against a hardcoded threat matrix, successfully dropping blacklisted vectors before memory allocation.
3. **Cryptographic Integrity Layer (`CryptoEngine`)**: Leverages `java.security.MessageDigest` to calculate deterministic 256-bit hexadecimal signatures (SHA-256) to ensure absolute data tamper-proofing.
